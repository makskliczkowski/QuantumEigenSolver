# Porting Artifacts

This directory stores Python-to-Julia port coordination artifacts.

## Layout

- `specs/`: module contracts produced by spec generation.
- `reports/port/`: implementation reports for each module.
- `reports/review/`: gate/review reports for each module.

## Naming

- File names follow module names (for example: `qes_solvers.md`).
- These files are documentation artifacts; source code lives in `juqusolver/`.
